HTML

HTML é uma linguagem para estruturação e apresentação de conteúdo para a World Wide Web

HTML 5 é a mais recente evolução do padrão que define a linguagem HTML

Esta versão traz importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade

Possibilita o uso de novos recursos antes possíveis apenas com a aplicação de outras tecnologias
Muitos recursos de HTML 5 têm sido construidos com a intenção de serem executados em dispositivos móveis como smartphones e tablets

História

Entre 1991 e 1999, ο HTML se desenvolveu a partir da versão 1 até a versão 4. Em 2008, a primera versão pública do HTML 5 foi lançada.

As primeiras versões do HTML foram definidas com regras vintáticas flexíveis, o que ajudou aqueles sem familiaridade com a Web. Com o tempo aumentou a tendencia em tornar a sintase cada vez mais rigida. Apesar disso, por questões históricas (retrocompatibilidade), os navegadores ainda hoje conseguem interpretar páginas web desenvolvidas com as primeiras versões do HTML.
